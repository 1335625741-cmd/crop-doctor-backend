# Diagnosis Prompt Template

## 基础 Prompt(单图或多图通用版)

```
你是一名经验丰富的农作物病害诊断专家,服务对象是普通农户。请根据用户上传的作物照片,综合判断可能的问题(病害 / 虫害 / 缺素症 / 药害 / 生理障碍),并按指定 JSON 结构输出。无法判断的字段填 null,并在 "uncertainty_reason" 中说清楚卡在哪里。

## 用户提供的上下文

(以上参考图来自本地 D:\\作物病害图\\ 的同病历史归档,描述由 GLM-4V 预生成;**仅作诊断参考**,如果与用户照片有冲突,以用户照片为准)

## 关键要求
- 严格按 JSON 输出,不要 markdown 包裹,不要多余说明文字
- 多张图给的是不同部位/不同角度时,综合判断,不要分别给结论
- 给出 2-3 个候选诊断时按可能性排序,每个标注置信度
- **每个候选必须给具体概率(`probability` 字段,0.00-1.00 之间,保留 2 位小数)**:基于图像视觉特征 + 用户提供的上下文(作物/天气/部位/用药史)综合判断。多个候选的概率之和应近似等于 1.00(归一化思维,Top 1 应明显高于 Top 2)
- 农户视角:原因和处置建议要白话、可执行,不要堆学术名词
- 不要给出具体农药商品名,只给通用类别(如"保护性杀菌剂"、"铜制剂"),让用户自己看当地农资店
- 不要在 immediate_actions 中给出可能危害的操作(如不戴手套直接接触药剂)

- 一个候选一个病,不要用"/"或"、"把多种病合并成一个诊断名
```

## 字段填充规则

| 占位符              | 填什么                                  | 没填时                       |
|----------------------|------------------------------------------|------------------------------|
| `{{crop_or_null}}`   | 用户报出的作物名(水稻/番茄/玉米…)        | 字符串 `null`               |
| `{{duration_or_null}}` | 出现症状多久(3 天/1 周/半个月)          | 字符串 `null`               |
| `{{weather_or_null}}`  | 近期天气(连阴雨/高温干旱/倒春寒)        | 字符串 `null`               |
| `{{chemical_or_null}}` | 最近用过什么药/肥(草甘膦/复合肥/…)      | 字符串 `null`               |
| `{{image_count}}`     | 数字,实际图片数量                       | 必填                         |
| `{{parts_or_null}}`   | 各张照片分别拍的是什么部位               | 不填或写 `null`             |
| `{{rag_references}}`  | RAG 检索出的 top-K 参考图(纯文字描述,无图)。**与用户照片不一致时,以用户照片为准**。启用 `--rag-lookup` 后注入,默认空字符串 | 字符串(可能为空) |
| `{{rag_candidates}}`  | (2026-07-20 加) RAG 检索出的 top-K 候选图,结构化 markdown 表格(每行:rank / 作物 / 病名 / rag_score / 描述),跟 `{{rag_references}}` 配合使用。启用 `bin/diagnose_with_rag.py` 后注入,默认空字符串 | 字符串(可能为空) |

## 多图部位建议

为提高识别准确度,推荐用户拍这些组合:

- **叶部病害**:受害叶正面 + 叶背(很多病斑、虫卵在背面) + 整体 + 特写
- **果实问题**:全果 + 病斑特写 + 邻近枝叶(看是否已传染)
- **全株萎蔫/黄化**:全株 + 根颈部(扒开土) + 茎基部 + 病叶
- **根部问题**:挖出根系 + 根瘤/腐烂特写 + 地上部对应症状

## 多图识别策略

当用户上传 2 张以上图片时,模型需要把不同部位/角度的信息综合成一个结论,而不是分别给每个部位下结论。具体要求:

1. **在 prompt 里明确 `parts_or_null`**:告诉模型每张图分别是什么部位(叶正面/叶背/全株/果实等),不要让模型自己猜
2. **综合判断,不要分别给诊断**:从所有图综合出 Top 1-3 候选,不要"图 1 是 A 病,图 2 是 B 病"
3. **优先信任特写**:如果全株看不清楚、特写很清晰,以特写为准
4. **冲突时的优先级**:特写 > 叶背 > 整体(因为特写细节最多,叶背有早期病征,整体只能看分布)
5. **互相印证**:如果多张图都显示同一症状,可提高置信度(高→极高);如果不同图显示不同症状(如叶背有霉层但叶面无斑),可能是复合病害,要在 `key_visual_clues` 里列出
6. **缺失部位的处理**:用户只拍了叶面没拍叶背时,在 `uncertainty_reason` 里标注"未见叶背,可能有早期症状未发现"

## 注意事项

- 不修改基础 prompt 的骨架,只做字段替换
- 不要在 prompt 里加"以下是图片"这类冗余句,模型自己会看到
- 多图时不要在 prompt 里写"这是第一张…这是第二张…",让模型自己看
- 如果用户只上传了 1 张不清晰的图,先按这个 prompt 调,再根据结果引导补拍

## ⚠️ 高混淆虫害视觉提示(2026-07-17 加,虫害独立保留后的诊断支撑)

诊断虫害时一定要看清**虫体 + 聚集部位 + 为害状**,以下是几组容易混淆的小型害虫(都是体长 1-3 mm 量级,LLM 一眼分不清),务必对比图像确认:

| 害虫 | 关键视觉特征(虫体+部位+为害状) | 易误判为 |
|---|---|---|
| **蚜虫** | **虫体**:梨形,长 1-2 mm,**腹管**(腹部末端两根小管)明显;有翅/无翅两型,体色多变(绿/黄/黑/红/带斑)。**聚集**:嫩梢、嫩叶背面、叶柄,**成群密集**,常上千头。**为害状**:叶片皱缩、卷曲、新叶畸形;分泌物(蜜露)致煤污 | 误认为"白粉虱/烟粉虱"(后者成虫有翅、白色蜡粉、停叶背不动)、"红蜘蛛"(后者蛛形纲、8 足、叶背结网)、"木虱"(后者成虫跳跃式、善跳) |
| **红蜘蛛(叶螨)** | **虫体**:极小 0.3-0.5 mm,**红色或黄绿色圆点**,8 足(蛛形纲,非昆虫)。**聚集**:叶背主脉两侧,**肉眼难见**;严重时叶面可见密集红/黄小点+丝网。**为害状**:叶片正面初现密集黄白色小斑点→连片枯黄→焦枯脱落(干旱季节暴发) | 误认为"锈病"(后者病斑橙黄粉状物、不动)、"蚜虫"(后者明显有虫体、腹管、6 足) |
| **白粉虱** | **虫体**:成虫 1-1.5 mm,**全体覆盖白色蜡粉**,翅 2 对白色膜翅,静止时翅屋脊状;若虫扁平椭圆形,贴叶背不动。**聚集**:叶背,**成虫善飞**,被扰动后成群起飞如"小白蛾"。**为害状**:叶面褪绿发黄、蜜露→煤污 | 误认为"烟粉虱"(同属,烟粉虱成虫略小、翅少蜡粉、寄主广)、"蚜虫"(后者梨形有腹管、停叶背) |
| **烟粉虱** | **虫体**:成虫 0.8-1.0 mm,较白粉虱小,**蜡粉薄**,翅半透明;B 型(MEAM1)更小更黄。**聚集**:叶背,可与白粉虱**混合发生**。**为害状**:除褪绿煤污外,B 型常致**番茄银叶**(叶面皱缩银白)、**西葫芦银叶**;Q 型抗药性极强 | 误认为"白粉虱"(后者大、白粉厚)、"蚜虫"(后者梨形有腹管) |
| **蓟马** | **虫体**:细长 1-2 mm,**翅 2 对缨翅**(羽毛状边缘),体色黄/黑/棕;**锉吸式口器**(右上颚退化,左锉吸);幼虫无翅白色/淡黄。**聚集**:花器内(花瓣/花蕊)、嫩叶未展开处、果实萼洼。**为害状**:花瓣褐斑畸形、嫩叶银白斑点(表皮被锉吸)、果实木栓化粗糙疤(果农称"花皮") | 误认为"蚜虫"(后者梨形腹管、停叶背)、"螨类"(后者圆点叶背结网) |
| **菜青虫(菜粉蝶幼虫)** | **虫体**:长 2-3 cm,**绿色纺锤形**,体表有细短绒毛,背线淡黄。**聚集**:十字花科叶背啃食,**啃出大缺刻**或孔洞,粪便绿色颗粒状。**为害状**:叶肉被啃成网状或大孔,严重时只剩叶脉 | 误认为"斜纹夜蛾幼虫"(后者体侧有明显**黄色纵线/三角形黑斑**、昼伏夜出)、"棉铃虫"(后者体色多变有刺) |

**虫害诊断流程建议**:
1. **先看虫体大小**:能不能看到虫?(红蜘蛛、蓟马若虫肉眼难见,要靠"为害状反推";蚜虫/菜青虫/斜纹夜蛾幼虫肉眼可见)
2. **再看 6 足还是 8 足**:**昆虫纲(6 足)**=蚜虫/粉虱/蓟马/菜青虫/斜纹夜蛾/木虱;**蛛形纲(8 足)**=红蜘蛛(不是昆虫!)
3. **看运动方式**:**跳**=木虱/叶蝉;**飞(白蜡粉)**=粉虱;**爬(密集)**=蚜虫;**静止/不动**=红蜘蛛/粉虱若虫;**夜出暴食**=斜纹夜蛾
4. **看聚集部位**:**嫩梢/嫩叶**=蚜虫/木虱;**叶背主脉**=红蜘蛛;**叶背(飞起如小白蛾)**=白粉虱;**花器/嫩叶未展开处**=蓟马;**叶面啃缺刻**=菜青虫/斜纹夜蛾
5. **看为害状**:**蜜露+煤污**=蚜虫/木虱/粉虱;**叶背丝网**=红蜘蛛;**银叶/银白斑**=蓟马/烟粉虱 B 型;**嫩叶皱缩**=蚜虫/木虱/烟粉虱;**啃成大孔/蛀果**=菜青虫/斜纹夜蛾

**诊断名规范**(同病害):
- 一个候选一个害虫,不要用"/"或"、"把多种虫合并
- 如果同时看到多种虫(常见:蚜虫+白粉虱;或蚜虫+红蜘蛛),作为**两个独立候选**列出
